fonction didentification de format, elle prend en parametre le type a convertir (convert), et la struc arguments :
dans cette fonction on declare un counteur qui sera return et est le nombre de char afficher sur la stdout

les types que je dois gerer avec les values que jenvoie a chaque fonction de conversion en utilisant va_arg(arguments, type);

'c' == int; == putchar (omg)
'p' == void * == ptr conversion (voir print memory)
's' == char * == putstr classique
'd' ou 'i' == int = base 10 conversion
'u' == unsigned int : unsigned int conversion
'x' == unsigned int : puthexa lowercase
'X' == unsigned int : puthexa uppercase

if (convert == 'c')
